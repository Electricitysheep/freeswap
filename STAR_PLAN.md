## 五、"训练龙虾"集成路线

结合用户的项目场景：

```
FreeSwap ←→ 龙虾 (Crawlee 数据挖掘系统)
  │
  ├─ 简单任务 → Groq (315 tok/s, 零成本)
  │   └─ 爬虫数据清洗、格式化、去重
  │
  ├─ 中等任务 → Gemini 2.5 Flash (1M 上下文)
  │   └─ 长文档摘要、数据结构化提取
  │
  ├─ 代码任务 → Mistral Codestral (最佳免费代码模型)
  │   └─ 爬虫脚本自动生成、修复
  │
  └─ 本地任务 → Ollama (完全离线)
      └─ 敏感数据本地处理
```

**关键价值**：让龙虾在零成本下 7x24 运行，自动在不同提供商之间切换。
